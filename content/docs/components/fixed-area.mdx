---
title: Fixed Area
description: A fixed area container with absolute positioning and centered content
---

## Installation

```bash
npx shadcn@latest add @shoscn/fixed-area
```

## Usage

```tsx
import { FixedArea } from '@/components/ui/layouts/fixed-area';

export default function MyComponent() {
  return (
    <FixedArea>
      <div>Centered Content</div>
    </FixedArea>
  );
}
```

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `React.ReactNode` | - | Child elements to display centered in the fixed area |
| `className` | `string` | - | Additional CSS classes to apply to the inner container |

## How It Works

The component creates a two-layer structure:
1. Outer layer: `relative flex-1` - Takes up available space
2. Inner layer: `absolute inset-0 flex items-center justify-center` - Absolutely positioned to fill parent and center content

## Examples

### Basic

```tsx
import { FixedArea } from '@/components/ui/layouts/fixed-area';

export default function Example() {
  return (
    <FixedArea>
      <div className="text-center">
        <h1>Centered Title</h1>
        <p>This content is perfectly centered</p>
      </div>
    </FixedArea>
  );
}
```

### With Custom Styling

```tsx
import { FixedArea } from '@/components/ui/layouts/fixed-area';

export default function Example() {
  return (
    <FixedArea className="bg-gradient-to-br from-blue-500 to-purple-600">
      <div className="text-white">
        <h1>Stylized Container</h1>
      </div>
    </FixedArea>
  );
}
```
