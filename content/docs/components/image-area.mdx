## Installation

```bash
npx shadcn@latest add @shoscn/image-area
```

This component automatically installs its dependency `@shoscn/fixed-area`.

## Usage

```tsx
import { ImageArea } from '@/components/ui/layout/image-area';

export default function MyComponent() {
  return (
    <ImageArea src="/background.jpg">
      <div>Content over blurred background</div>
    </ImageArea>
  );
}
```

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `React.ReactNode` | - | Child elements to display over the blurred background |
| `src` | `string` | - | URL of the background image |
| `className` | `string` | - | Additional CSS classes to apply to the FixedArea container |

## How It Works

The component creates a layered effect:
1. Background layer: Absolutely positioned, scaled (110%), heavily blurred (3xl), and semi-transparent (30%)
2. Content layer: Uses `FixedArea` to center content with padding

## Features

- Blurred background image effect
- Scaled background for better edge coverage
- Centered content overlay
- Customizable opacity and blur
