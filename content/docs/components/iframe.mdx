```bash

```

## Usage

```tsx
import IFrame from '@/components/ui/utils/iframe';

export default function MyComponent() {
  return (
    <div className="w-full h-screen">
      <IFrame title="My App" url="https://example.com" />
    </div>
  );
}
```

### Helper Function

The component also exports a `createIFrameRoute` helper function for creating route components.

```tsx
import { createIFrameRoute } from '@/components/ui/utils/iframe';

// Create a route component for a local server
const MyServerRoute = createIFrameRoute('My Server', 3000);

export default MyServerRoute;
```

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `title` | `string` | - | Required title for the iframe (for accessibility) |
| `url` | `string` | - | URL to load in the iframe |
| `port` | `number` | - | Local port number (constructs URL from current hostname) |

## Notes

- Either `url` or `port` should be provided
- If `port` is used, the iframe will load from the current hostname on that port
- The iframe includes `clipboard-read` and `clipboard-write` permissions
- The component fills its parent container (100% width and height)
