```bash
npx shadcn@latest add @shoscn/flex
```

## Usage

```tsx
import { Flex } from '@/components/ui/layout/flex';

export default function MyComponent() {
  return (
    <Flex row justify="center" align="center" className="gap-4">
      <div>Item 1</div>
      <div>Item 2</div>
      <div>Item 3</div>
    </Flex>
  );
}
```

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `row` | `boolean` | `false` | Use row direction (default is column) |
| `reversed` | `boolean` | `false` | Reverse the flex direction |
| `justify` | `'start' \| 'end' \| 'center' \| 'between' \| 'around' \| 'evenly'` | `'start'` | Justify content along the main axis |
| `align` | `'start' \| 'end' \| 'center' \| 'baseline' \| 'stretch'` | `'stretch'` | Align items along the cross axis |
| `withPaddings` | `boolean` | `false` | Apply CSS variable-based padding and gap |
| `expanded` | `boolean` | `false` | Apply `flex-1` to fill available space |
| `className` | `string` | - | Additional CSS classes |
