## Installation

```bash
npx shadcn@latest add @shoscn/spinner
```

## Usage

```tsx
import { Spinner } from '@/components/ui/utils/spinner';

export default function MyComponent() {
  return <Spinner />;
}
```

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes for custom styling |
| All SVG props | - | - | Accepts all standard SVG element props |

## Accessibility

- The spinner includes `role="status"` for screen readers
- Includes `aria-label="Loading"` to describe the loading state
